Task Management System:
This is a Python program that allows users to register, log in, and manage tasks assigned to them.

Getting Started:
The following instructions will help you run the program on your local machine.

Prerequisites:
Python 3.x
A command-line interface

Installation:
Clone the repository or download the source code as a ZIP file.
Extract the files to your preferred directory.

Usage:
Open your command-line interface.
Navigate to the directory containing the source code.
Run the command python task_management_system.py to start the program.
Follow the prompts to register a new user or log in.
Once logged in, you will be presented with a menu of options to manage your tasks.

Functionality:
Register a new user
Log in with an existing username and password
Add a new task
View all tasks
View only tasks assigned to the logged-in user
Edit a task by marking it as complete or changing the assigned user
